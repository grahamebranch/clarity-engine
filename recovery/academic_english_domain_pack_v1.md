# Academic English Domain Pack (v1) — Clarity Engine

## 1. Purpose
The Academic English Domain Pack defines the rules, templates, flows, and vocabulary required to generate CEFR‑aligned academic communication lessons.

This is a Phase‑4 extension and does not modify:
- system invariants
- lesson structure
- CEFR rules
- governance
- edition isolation

---

## 2. Domain Identity

### 2.1 Name
**Academic English Domain**

### 2.2 Scope
Covers academic communication in:
- essays
- reports
- presentations
- research summaries
- academic discussions
- critical analysis
- referencing and citation language

### 2.3 CEFR Range
B1 → C2  
(A‑levels are not included due to cognitive load.)

---

## 3. Domain Rules

### 3.1 Domain Invariants
Academic English lessons must:
- follow the global lesson structure
- include academic vocabulary
- include formal register
- include critical‑thinking tasks
- include structured argumentation
- avoid colloquial language

### 3.2 Domain Restrictions
Academic English lessons must NOT:
- include real research data
- include copyrighted academic texts
- include political or ideological persuasion
- include university‑specific requirements

### 3.3 Domain Alignment
All tasks must be:
- formal
- structured
- evidence‑based
- CEFR‑appropriate

---

## 4. Domain Templates

### 4.1 Essay Template
Includes:
- thesis statement
- supporting arguments
- counter‑argument
- conclusion
- academic connectors

### 4.2 Report Template
Includes:
- introduction
- methodology (simplified)
- findings
- discussion
- recommendations

### 4.3 Academic Presentation Template
Includes:
- research question
- structure overview
- data commentary
- implications
- Q&A

### 4.4 Summary Template
Includes:
- main idea
- key points
- supporting evidence
- neutral tone

---

## 5. Domain Flows

### 5.1 Essay Writing Flow
1. Analyse the question  
2. Generate thesis  
3. Build outline  
4. Write paragraph set  
5. Review coherence  

### 5.2 Report Writing Flow
1. Identify purpose  
2. Select structure  
3. Draft sections  
4. Add academic register  
5. Finalise  

### 5.3 Academic Discussion Flow
1. Introduce position  
2. Provide evidence  
3. Respond to counter‑points  
4. Summarise

## 7. Functional Language Sets

### 7.1 Argumentation
- “The evidence suggests that…”  
- “A key implication is…”  
- “This supports the claim that…”  

### 7.2 Critical Analysis
- “A limitation of this approach is…”  
- “An alternative interpretation is…”  

### 7.3 Summarising
- “In summary…”  
- “The main findings indicate…”

Academic English lessons must include:
- citation norms

## 9. Validation Rules

Academic English outputs must:
- follow CEFR  
- follow edition rules  
- avoid bias  
- avoid ideological persuasion

- real research data  
- political advocacy  
- personal data retention
