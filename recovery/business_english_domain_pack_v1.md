# Business English Domain Pack (v1) — Clarity Engine

## 1. Purpose
This domain pack defines the **rules, templates, flows, vocabulary sets, and functional language** required to generate Business English lessons inside the Clarity Engine.

It extends the engine without modifying:
- system invariants  
- lesson structure  
- edition rules  
- CEFR alignment  
- governance  

This is a **Phase‑4 extension**, not a system‑layer change.

---

## 2. Domain Identity

### 2.1 Name
**Business English Domain**

### 2.2 Scope
Covers professional communication in:
- meetings  
- presentations  
- negotiations  
- email writing  
- workplace interactions  
- reporting  
- project communication  
- customer communication  

### 2.3 CEFR Range
A2 → C2  
(With different templates and flows per level.)

---

## 3. Domain Rules (Authoritative)

### 3.1 Domain Invariants
Business English lessons must:
- follow the global lesson structure  
- include functional language  
- include workplace‑relevant vocabulary  
- include scenario‑based tasks  
- include cultural considerations (global business norms)  
- avoid industry‑specific jargon unless level‑appropriate  

### 3.2 Domain Restrictions
Business English lessons must NOT:
- include confidential corporate content  
- include real company data  
- include political or legal advice  
- include financial forecasting  

### 3.3 Domain Alignment
All tasks must be:
- realistic  
- workplace‑aligned  
- communication‑focused  
- CEFR‑appropriate

## 5. Domain Flows

### 5.1 Meeting Skills Flow
1. Warm‑up: workplace scenario  
2. Presentation: meeting language  
3. Controlled practice: scripted dialogue  
4. Freer practice: roleplay  
5. Assessment: summary + action points  

### 5.2 Email Writing Flow
1. Identify purpose  
2. Select tone  
3. Build structure  
4. Draft email  
5. Improve clarity  
6. Final version  

### 5.3 Presentation Skills Flow
1. Topic selection  
2. Structure planning  
3. Slide language  
4. Delivery practice  
5. Q&A simulation

## 7. Functional Language Sets

### 7.1 Meetings
- “Can we move to the next point…”  
- “I’d like to add something…”  
- “Could you clarify what you mean by…”  

### 7.2 Emailing
- “I’m writing to follow up on…”  
- “Please find attached…”  
- “Could you confirm…”  

### 7.3 Presentations
- “Let’s begin by looking at…”  
- “As you can see from this chart…”  
- “To summarise…”  

### 7.4 Negotiations
- “We could consider an alternative…”  
- “What if we adjust the timeline…”  
- “Our priority is…”
