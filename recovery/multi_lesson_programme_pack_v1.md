# Multi‑Lesson Programme Pack (v1) — Clarity Engine

## 1. Purpose
The Multi‑Lesson Programme Pack defines the structures, rules, and templates required to generate **sequenced learning programmes** rather than single lessons.

This is a Phase‑4 extension and does not modify:
- system invariants
- lesson structure
- CEFR rules
- governance
- edition isolation

---

## 5. Lesson Sequence Rules

### 5.1 Sequencing Logic
Lesson sequences must:
- increase complexity gradually  
- recycle vocabulary  
- reinforce functional language  
- alternate receptive/productive skills  
- include periodic consolidation  

### 5.2 Programme Length Rules
- 4‑lesson modules: one skill focus  
- 8‑lesson programmes: two skill focuses  
- 12‑lesson programmes: full skill cycle  

### 5.3 Assessment Placement
- short programmes: final assessment only  
- medium programmes: mid + final  
- long programmes: diagnostic + mid + final  

---
